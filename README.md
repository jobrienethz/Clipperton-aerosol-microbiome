# Clipperton-aerosol-microbiome
Repository associated with the manuscript "Daytime Enrichment of Marine Microbes in the Airborne Microbiome follows the Diel Cycle of Sea Spray Aerosol Production"

It contains the code used to decontaminate, analyse and visualise the results and figures based on the data available at Zenodo

Link to manuscript is to follow.

# Structure
> Resources/ metadata and tables used by scripts

> Code/ the R scripts used to decontaminate, analyse and visualise the results and figures
